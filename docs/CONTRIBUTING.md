# Contributing

Thanks for helping improve this project.

## Getting Started

1. Fork the repository and create a branch from `main`.
2. Keep changes focused and small where possible.
3. Run `./install.sh` and verify conversion works locally.
4. Update docs when behavior or usage changes.

## Pull Request Checklist

- Explain what changed and why.
- Include before/after examples when relevant.
- Keep scripts executable when required (`chmod +x`).
- Ensure no unrelated files are changed.

## Reporting Issues

Please include:

- Steps to reproduce
- Expected behavior
- Actual behavior
- Environment details (OS, Pandoc version, Node version)
